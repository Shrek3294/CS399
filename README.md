# Student Engagement Dashboard

Frontend dashboard prototype for Northview University Student Success, built with React + Vite.

## Tech Stack
- React (functional components)
- Vite
- Plain CSS (`src/index.css`)

## Local Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in terminal (usually `http://localhost:5173`).

## Scripts
- `npm run dev` - Start local development server
- `npm run build` - Build production assets into `dist/`
- `npm run preview` - Preview production build

## Current Pages
- Dashboard (command center overview)
- Student Profile QuickView (with smart filters)
- At-Risk Alerts
- Advising Sessions
- Tutoring
- Reports
- Settings

## UI Notes
- Includes a light/dark mode toggle in the top-left sidebar header.
- Uses mock data placeholders with `TODO` comments for future backend integration.
